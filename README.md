\# Tomato Leaf Disease Detection 🍅



This project uses Deep Learning and Streamlit to detect diseases in tomato leaves.



\## Features

\- Upload tomato leaf image

\- Detect disease using AI model

\- Show disease reason and solution

\- Display prediction confidence



\## Technologies Used

\- Python

\- TensorFlow / Keras

\- Streamlit

\- NumPy

\- Pillow



\## How to Run



1\. Install requirements



pip install -r requirements.txt



2\. Run Streamlit app



streamlit run app.py



\## Dataset Classes



\- Tomato\_Early\_blight

\- Tomato\_Late\_blight

\- Tomato\_Leaf\_Mold

\- Tomato\_Septoria\_leaf\_spot

\- Tomato\_Spider\_mites

\- Tomato\_Target\_Spot

\- Tomato\_Yellow\_Leaf\_Curl\_Virus

\- Tomato\_mosaic\_virus

\- Tomato\_healthy
